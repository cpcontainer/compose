Help - Method

Method means which way the server is aquiring data from Dishy. Change inside congif.inc.php

BROWSER - Requires you to have a browser open and accessing the web app so that the server can read the stats. This is simple but can cause issues if too many tabs are open viewing this.

Speedtest will not operate using this method. The more reliable and consistent method is CLI.

CLI - Uses cron jobs and hidden screen sessions to update the server. This runs all the time and will collect data even if you are not viewing the page.

In the scripts/binbash folder you will find two .sh files. You can modify them (path), and place them on your system. Run each one in a screen session.

The speedtest is found in script/cron/php/speedtest.cron.php.
Add that to your cron job.
Ex: */5 * * * * php /your/install/path/here/speedtest.cron.php.

Please do not run every minute. Be kind to Sats and Servers
